USAGE of HATAKE:
-------------------------------------------------------------------

Cloning the repo:
```
git clone https://github.com/Keyj33k/Hatake.git
```
After cloning:
```
cd Hatake
```
Install python requirements:
```
pip3 install -r requirements.txt
```
Run the script:
```
python3 hatake.py
```
